‚ö° Essential Git Commands

Clone the Game Repo

git clone [repo-url]
cd Neon-Runner-Artifact-Heist

Create Your Branch

git checkout -b yourname-fixes

Make Changes & Commit

git add .
git commit -m "Fix: description of what you fixed"
git push origin yourname-fixes

Check History

git log --oneline
git log --graph --oneline --all

Find Bugs

git log --oneline | grep -i "keyword"
git show [commit-hash]

üéÆ Running the Game

Setup (First Time)

# Create virtual environment
python -m venv venv

# Activate it
# Mac/Linux:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

Run the Game

python main.py

Game Controls

  • Move: WASD or Arrow Keys
  • Quit: Q
  • Restart: R (after game over)

üÜò Common Issues

Can't Run the Game

  • Make sure virtual environment is activated
  • Install requirements: pip install -r requirements.txt
  • Windows users: Use WSL or Git Bash

Git Problems

  • Merge conflicts? Ask instructor for help
  • Can't push? Make sure you're on your branch
  • Lost changes? Check git status

Mission Board Issues

  • Mission locked? Someone else claimed it - choose another
  • Can't claim? Try refreshing the page
  • Lost progress? Your username stores your session
